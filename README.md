# 🚦 AI-Powered Road Sign Recognition System

Welcome to the Road Sign Recognition project! This repository contains the code and educational materials to build a deep learning model that can accurately detect and classify traffic signs. 

This project is designed not just as a functional AI model, but as an **educational tool** to introduce students to the fascinating world of Computer Vision and Autonomous Vehicles.

## 🎯 Project Objective
To understand how self-driving cars "see" the road by building a Convolutional Neural Network (CNN) that processes images of road signs and predicts their meaning in real-time.

## 🗂️ Project Structure
This project is broken down into 5 easy-to-follow steps:
* `step1_view_data.py`: Exploring and visualizing the dataset.
* `step2_preprocess.py`: Resizing and preparing images for the AI.
* `step3_train.py`: Building and training the neural network.
* `step4_realtime.py`: Testing the AI with new images or a webcam.
* `step5_dashboard.py`: A simple user interface to display results.
* `my_road_sign_model.keras`: The pre-trained AI model (ready to use!).

## 🛠️ Tools & Technologies Used
* **Python 3**
* **OpenCV (`cv2`)**: For computer vision and image processing.
* **TensorFlow / Keras**: For building the deep learning model.
* **NumPy & Pandas**: For data manipulation.
* **Matplotlib**: For visualizing the training progress.

## 🚀 How to Run the Project
1. Download or clone this repository to your computer.
2. Ensure you have the required libraries installed.
3. If you want to test the AI immediately, run `step4_realtime.py` or `step5_dashboard.py` (it will use the included `.keras` model).
4. If you want to learn how the AI learns, start from `step1` and work your way through!

*(Note: The raw image dataset is not included in this repository due to size limitations, but the pre-trained model is fully functional without it!)*
